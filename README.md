# 🚀 Bootstrapping a New Machine

This repo + [dotfiles-private](https://github.com/p4p3r/dotfiles-private) can fully set up a new macOS or Linux laptop using **Nix** + **chezmoi**, with secrets from **1Password**.

## Quick start

Run this one-liner:

```bash
bash -c "$(curl -fsSL https://gist.githubusercontent.com/p4p3r/9724833647dd3217414f4463e5ca52bb/raw/1227e8ef6b712bbdf619151a8bdca44f39f76103/bootstrap-new-machine.sh)"
```
