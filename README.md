# 🚀 Bootstrapping a New Machine

This repo + [dotfiles-private](https://github.com/p4p3r/dotfiles-private) can fully set up a new macOS or Linux laptop using **Nix** + **chezmoi**, with secrets from **1Password**.

## Quick start

Run this one-liner:

```bash
bash -c "$(curl -fsSL https://gist.githubusercontent.com/p4p3r/9724833647dd3217414f4463e5ca52bb/raw/5554c3ff6b24e3b875e964aaccf1da70379c60fc/bootstrap-new-machine.sh)"
```
